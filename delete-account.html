이 페이지는 Gotcha 사용자가 계정과 관련 데이터 삭제를 요청하는 방법을 안내합니다. 대부분의 경우 앱 내에서 즉시 삭제를 완료할 수 있습니다.

한국어(KO) English(EN)

앱 내 삭제(권장)

Gotcha 앱에서 아래 순서로 계정을 삭제할 수 있습니다.

  1. Gotcha 앱 실행
  2. 설정(Settings) 이동
  3. 계정 삭제(Delete Account) 선택
  4. 확인 절차를 진행하여 삭제 완료

앱 접속이 어려운 경우, 아래 “이메일 요청” 방법을 이용하세요.

Email deletion request (alternative)

If you cannot access the app, request deletion by email:

  • Email: review.gotcha@gmail.com
  • Subject: Account Deletion Request
  • Include the identifier used to sign in (Kakao / Google / Apple).

We may request additional information only to verify account ownership.

삭제되는 데이터

  • 프로필 정보(프로필 사진, 닉네임, 인스타그램 아이디, 한줄소개, 선택 입력 항목 등)
  • 서비스 이용을 위해 계정에 연결된 데이터(예: 코인 잔액/이용 기록 등, 해당되는 경우)
  • 사용자가 제출한 기타 콘텐츠(해당되는 경우)

처리 기간 및 예외 보관

삭제 요청 확인 후 가능한 한 신속히 처리합니다. 단, 법령 준수/보안/부정이용 방지/분쟁 해결을 위해 필요한 경우 일부 정보가 제한 기간 보관될 수 있으며, 해당 목적 달성 후 삭제합니다.

로그인 방식

Gotcha는 ID/비밀번호 로그인을 제공하지 않고 소셜 로그인만 지원합니다.
